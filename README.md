# Arda
**A photo-sharing server written in ~900 lines of Scala 3**

Its built on the [http4s](https://http4s.org/) stack, and deliberately avoids FR/ORM's in favour of a "Raw JDBC" style.

### Features
- JWT Auth
- Admin/user roles
- Customizable storage engines
